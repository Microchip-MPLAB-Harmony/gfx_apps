Description 
legato_monitor 
This application demonstrates the graphics capabilities of the legato graphics library by simulating a user interface for a blood pressure and heartbeat meter device.

 

legato_showcase 
This demonstration provides a subset of capabilities offered by the legato graphics library.

 

legato_benchmark 
This application demonstrates frame update rate metrics on the various rendering operations in the aria graphics library.

 

legato_quickstart_ext_res 

This demonstration provides a touch-enabled starting point for reading images, string, and multiple fonts from both internal and external non-volatile memory (NVM).

The legato_flash demonstration application serves as an external memory programmer to flash the off-chip non-volatile memory .
