The application demonstrates advanced graphics techniques such as parallax and sprite animation.

This application demonstrates frame update rate metrics on the various rendering operations in the aria graphics library.

This application demonstrates a simulated vehicle dashboard running on the SAM A5D2 MPU built using the aria graphics library. This application is for graphics demonstration purposes only and is not designed to work with safety-critical systems.

This application demonstrates the use of blit features of the graphics processor using large image files.
